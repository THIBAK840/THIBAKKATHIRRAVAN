# Thibakkathirravan H — Portfolio

Personal portfolio website built for GitHub Pages.

## Deploy on GitHub Pages
1. Upload `index.html`, `style.css`, `script.js`, and `resume.pdf` to the repository root.
2. Open **Settings → Pages** in GitHub.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Choose `main` and `/ (root)`, then **Save**.

The site should then be available at `https://thibak840.github.io/THIBAKKATHIRRAVAN/`.
